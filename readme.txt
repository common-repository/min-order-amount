Set a WooCommerce minimum order amount and change the notice texts for the cart & checkout pages.

== Description ==
Set a WooCommerce minimum order amount and change the notice texts for the cart & checkout page.
If minimum order value is not satisfied in checkout page then users can't checkout or place order.
This plugin will give you an advantage option for WooCommerce minimum order amount, as well as options to separately change the notice texts for the cart and checkout pages.

== Installation ==
1. Upload the plugin files to the '/wp-content/plugins' directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. GOTO to ***WooCommerce -> Settings*** and scroll down to the Minimum order settings to configure the options.

== Screenshots ==

1. Settings Options are available under **WooCommerce -> Settings**, at bottom of the page.
2. Notice example when the cart does not meet the minimum order amount.

== Frequently Asked Questions ==
= Does this work for WordPress < 5.0 ? =
If you are using a version before 5.0, It'll work but recommended to use latest Wordpress and WooCommerce.

= Does this work only in the WooCommerce ? =
Yes, this only works with WooCommerce.

== Suggestions ==
